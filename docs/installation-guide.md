# 導入手順

## ローカル検証

```powershell
cd D:\AI\VSCodeExtension\ui-text-translation-key-manager
npm test
npm start
```

## 配布前確認

- `dist/ui-text-translation-key-manager-docs.zip` を確認する。
- `docs/release-checklist.md` を完了状態にする。
- GitHub Release / VSIX に出す前に手動テストを実施する。
